Q2 优客找房预约指南

一、如何快速的在官网搜索到目标房源（首先进入官网www.uoko.com的首页）

1.条件找房

进入“我要租房”板块，选择区域、租金、户型等标签信息，缩小找房范围或使用我们的快捷搜索“地图找房”，直接在地图上寻找房源

 

2.查看房源

按区域、租金等条件查找后即可看到对应的房源


点击即可进入查看房屋具体信息


3.预约看房

方式一：直接拨打400-000-4170优客官网预约电话

方式二：点击“我要看房”，输入对应信息即可

方式三：点击“QQ咨询”，可以直接咨询客服小伙伴


二、找房注意事项

1.如何节省找房时间

   不同的优客逸家管家会负责不同的区域，所以在预约时请告知清楚需求，让管家帮忙推荐可以最大限度节省宝贵的时间

2.看房前考虑清楚对交通、户型、周边配套等要素的需求度，提高找房精准度

3.交付钥匙

   在合同规定的房屋交付日交验时您需注意：（1）房屋是否与合同约定的一致；（2）家电、家具等是否与合同相符；（3）钥匙是否已交付；（4）水电气表读书等（物业交割时未满会帮您计算清楚）


Q3 优客免息月付及费用计算方式

房租月付帮你完美解放钱包，各种面膜、美食向你袭来，当然享受月付还是会有些小小的条件：

1、享受月付需要满足签约两年及以上的条件哦（签约两年的福利请看“关于租期的选择”）

首次支付：押二付一 （毫无鸭梨耶~）
=月付租金的押金*2 +月付租金*1+ 月付租金的管理服务费*1
管理服务费按月定额收取，中途退租，退还当月剩余天数费用。
附加增配项目：如加装空调等，都单独收取。

------------------------------------------------------

以600元房子为例：

该房子的优客月付租金=600元
因此需首次支付：
月付押金：600元*2个月=1200元
月付租金：600元*1个月=600元
月付租金的管理服务费定额收取100元
共计：1900元

------------------------------------------------------

支付方式： 现场：与签约管家用POS机支付

------------------------------------------------------

后续支付：剩余23个月房租，按月按时存入自己绑定的银行卡即可！

注意昂：关于月付的细节，可以详询400-000-4170
满足两年起租条件的，才能选择月付哦！！！中途解约除了正常的违约金外，还将有少量的月付违约手续费（违约手续费是银行收取的···）

月付违约手续费收取规则：

入住未满一年，将收取（1个月房租+1个月服务管理费）*20%

若房租为600元/月，则为（600+100）*20%=140元的月付违约金

入住1-2年，将收取（1个月房租+1个月服务管理费）*10%

若房租为600元/月，则为（600+100）*10%=70元的月付违约金

二、房间签约


1、交定后，以交定时间点开始计时，于交定的3天后开始计算租期（若是未交付房源，以交付时间点开始计时，于交付的3天后开始计算租期）。

即，交定后的3天（72小时）内务必与公司签约并支付尾款，否则定金作废！公司有权对交定房间再次出租。

       举个栗子：

如果是3月1日24:00前完成定金交付，那么，您务必在3月3日24:00前与公司签订租房合同，否则定金作废，且不退不换！

2、如有特殊情况，需要提前沟通，在双方都同意的情况下+已完成首次房租支付+租期已经从交定后第3天开始计算的三个前提下，可以延迟到交定完成后的7天内签写租房合同。

3、友情提示：房间以交定为准，逾期则定金不退不换！
